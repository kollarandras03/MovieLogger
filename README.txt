[This project is working progress]

GOALS:
Make a desktop application that is capable of managing a real life collection of movies making it easier 
to check, wether a DVD is already in the collection. 

PLANNED FEATURES:
- it is possible to store DVD-s using properties like: title, actor(s), genre etc.
- it is possible to search for a movie using the properties defined above
- it is possible to add a comment to a movie, for easier searching
